# - Python - Programação Orientada a Objetos

Este repositório contém meus estudos e exercícios de **Programação Orientada a Objetos (POO) em Python**.

Os códigos fazem parte do meu aprendizado em desenvolvimento de software, abordando conceitos fundamentais da orientação a objetos e integração com banco de dados.

## - Conteúdos estudados

* Classes
* Objetos
* Métodos
* Encapsulamento
* Herança
* Polimorfismo
* Modularização
* Organização de projetos
* Integração com banco de dados SQLite

## - Objetivo

Praticar e consolidar os conceitos de **POO em Python**, desenvolvendo códigos organizados, reutilizáveis e mais próximos das práticas utilizadas no desenvolvimento profissional.

## - Estrutura

Os arquivos estão organizados de acordo com as aulas, exemplos e exercícios realizados durante o curso.
